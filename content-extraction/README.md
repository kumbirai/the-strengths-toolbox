## Extraction Template

Use the following template for each content file:

```markdown
# [Page/Section Title]

## URL
[Source URL]

## Page Structure
- H1: [Main heading]
- H2: [Section headings]
- H3: [Subsection headings]

## Content Sections

### Section 1: [Section Name]
[Content text]

## CTAs
- [CTA text] → [Link destination]

## Links
- [Link text] → [URL]

## Images
- [Image description] → [Image URL or location]

## Notes
[Any additional notes about formatting, special elements, etc.]
```
