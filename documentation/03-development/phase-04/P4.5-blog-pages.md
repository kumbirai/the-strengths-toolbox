# P4.5 Blog Pages Implementation Plan

## 1. Overview and Objectives

### 1.1 Purpose
This document provides detailed step-by-step instructions for implementing blog listing, detail, category, tag, and search pages.

### 1.2 Scope
This implementation plan covers tasks P4.5.1 through P4.5.6:
- **P4.5.1**: Build blog listing page
- **P4.5.2**: Build blog post detail page
- **P4.5.3**: Implement pagination
- **P4.5.4**: Build category archive page
- **P4.5.5**: Build tag archive page
- **P4.5.6**: Implement search functionality

### 1.3 Success Criteria
- Blog listing displays posts correctly
- Post detail page shows full content
- Pagination works
- Category/tag archives function
- Search returns relevant results
- SEO metadata configured
- Responsive design works

## 2. Prerequisites

### 2.1 Dependencies
- Task P2.3.3 completed (BlogController created)
- Task P2.1.2 completed (BlogPostService created)
- BlogPost model and relationships functional

### 2.2 Reference Documents
- Frontend Architecture: `documentation/01-architecture/04-frontend-architecture.md`

## 8. Task P4.5.6: Implement Search Functionality

### 8.1 Update BlogController

**File: `app/Http/Controllers/Web/BlogController.php`** (add/update method)

```php
public function search(Request $request)
{
    $query = $request->get('q');
    
    if (!$query) {
        return redirect()->route('blog.index');
    }

    $posts = $this->blogPostService->search($query, 12);
    
    return view('blog.search', [
        'posts' => $posts,
        'query' => $query
    ]);
}
```

### 8.3 Update BlogPostService

**File: `app/Services/BlogPostService.php`** (add search method)

```php
public function search(string $query, int $perPage = 12)
{
    return BlogPost::where('is_published', true)
        ->where('published_at', '<=', now())
        ->where(function($q) use ($query) {
            $q->where('title', 'like', "%{$query}%")
              ->orWhere('excerpt', 'like', "%{$query}%")
              ->orWhere('content', 'like', "%{$query}%");
        })
        ->orderBy('published_at', 'desc')
        ->paginate($perPage);
}
```
