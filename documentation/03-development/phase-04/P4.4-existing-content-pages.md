# P4.4 Existing Content Pages Implementation Plan

## 1. Overview and Objectives

### 1.1 Purpose
This document provides detailed step-by-step instructions for implementing all existing content pages from the current The Strengths Toolbox website, including Strengths-Based Development, Sales Training, Facilitation/Workshop pages, and standalone pages.

### 1.2 Scope
This implementation plan covers tasks P4.4.1 through P4.4.7:
- **P4.4.1**: Create Strengths-Based Development pages
- **P4.4.2**: Create Sales Training pages
- **P4.4.3**: Create Facilitation/Workshop pages
- **P4.4.4**: Create Keynote Talks page
- **P4.4.5**: Create Books page
- **P4.4.6**: Create Testimonials listing page
- **P4.4.7**: Create Privacy Statement page

### 1.3 Success Criteria
- All pages render correctly
- Navigation structure works
- Content is accessible via CMS
- SEO metadata configured
- Responsive design works

### 2.1 Dependencies
- Task P2.3.2 completed (PageController created)

### 3.2 Step-by-Step Implementation

#### Step 1: Create Reusable Page Template Component

**File: `resources/views/components/content-page.blade.php`**

```blade
@props(['page', 'showBreadcrumbs' => true])

@if($showBreadcrumbs)
    <nav class="bg-gray-100 py-4" aria-label="Breadcrumb">
        <div class="container-custom">
            <ol class="flex items-center space-x-2 text-sm">

                <li class="text-gray-700">{{ $page->title }}</li>
            </ol>
        </div>
    </nav>
@endif

<section class="section-padding bg-white">
    <div class="container-custom">
        <article class="max-w-4xl mx-auto">
            <header class="mb-8">
                <h1 class="text-4xl md:text-5xl font-bold text-gray-900 mb-4">
                    {{ $page->title }}
                </h1>
                @if($page->excerpt)
                    <p class="text-xl text-gray-600">{{ $page->excerpt }}</p>
                @endif
            </header>

            <div class="prose prose-lg max-w-none">
                {!! $page->content !!}
            </div>

            @if($page->cta_text && $page->cta_link)
                <div class="mt-12 text-center">
                    <a href="{{ $page->cta_link }}" class="btn btn-primary text-lg">
                        {{ $page->cta_text }}
                    </a>
                </div>
            @endif
        </article>
    </div>
</section>
```

#### Step 2: Update Page Show View

**File: `resources/views/web/pages/show.blade.php`**

```blade
@extends('layouts.app')

@section('title', $page->meta_title ?? $page->title)

@push('meta')
    <meta name="description" content="{{ $page->meta_description ?? $page->excerpt }}">
    @if($page->meta_keywords)
        <meta name="keywords" content="{{ $page->meta_keywords }}">
    @endif
    
    @if($page->seo)
        <meta property="og:title" content="{{ $page->seo->og_title ?? $page->title }}">
        <meta property="og:description" content="{{ $page->seo->og_description ?? $page->excerpt }}">
        <meta property="og:type" content="website">
        <meta property="og:url" content="{{ url('/' . $page->slug) }}">
        @if($page->seo->og_image)
            <meta property="og:image" content="{{ $page->seo->og_image }}">
        @endif
    @endif
@endpush

@section('content')
    <x-content-page :page="$page" />
@endsection
```

```blade
@extends('layouts.app')
```

```blade
@push('meta')
    <meta name="description" content="Read testimonials from businesses that have transformed their teams with The Strengths Toolbox.">

                </p>

            @if($testimonials->count() > 0)

                                            <path d="M9.049 2.927c.3-.921 1.603-.921 1.902 0l1.07 3.292a1 1 0 00.95.69h3.462c.969 0 1.371 1.24.588 1.81l-2.8 2.034a1 1 0 00-.364 1.118l1.07 3.292c.3.921-.755 1.688-1.54 1.118l-2.8-2.034a1 1 0 00-1.175 0l-2.8 2.034c-.784.57-1.838-.197-1.539-1.118l1.07-3.292a1 1 0 00-.364-1.118L2.98 8.72c-.783-.57-.38-1.81.588-1.81h3.461a1 1 0 00.951-.69l1.07-3.292z"/>

                                        {{ strtoupper(substr($testimonial->name, 0, 1)) }}

                                </div>
```
