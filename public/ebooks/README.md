# eBook Directory

This directory contains the free eBook file for download.

## Placeholder File

The file `free-ebook.pdf` is a placeholder that should be replaced with the actual eBook PDF file.

## File Location

The eBook file should be placed at: `public/ebooks/free-ebook.pdf`

## Configuration

The file path is configured in `config/ebook.php` and can be overridden via the `EBOOK_FILE_PATH` environment variable.
